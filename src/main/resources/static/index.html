Welcome to Price Watcher!

Guidelines to use application:

  1. Move to navigation menu on the left
  2. Check price lists, dates, brands, products and prices preloaded for you
  3. The table "Prices" is presented in the form of cards set for each price item
  4. You can also create, edit and delete all the tables
  5. In the menu point "Prices" you can use the filters at the top to select prices you need
